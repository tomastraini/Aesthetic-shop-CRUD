# Aesthetic-shop-CRUD
This is a high-performance, simple-shortcut and easy to use program made by me to upload all the information needed for an aesthetic shop, to be aware:
- Clients CRUD
- Products CRUD
- Turns CRUD
- Queries CRUD
- Medical files CRUD
- Products sales CRUD
- Users CRUD (Counts with permissions)
- Habits CRUD
- Treatments CRUD
Enjoy!
